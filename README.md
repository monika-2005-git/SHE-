# SHE-

## Problem Statement
Students often face confusion while choosing the right career domain based on their interests, skills, and academic background.
Even after selecting a domain, they lack structured guidance on preparation strategy, required skills, and suitable companies,
leading to inefficient preparation for campus placements.

## Proposed Solution
SHE- is an AI-powered student career guidance platform that:
- Recommends suitable career domains
- Analyzes resumes for role alignment
- Identifies skill gaps
- Provides personalized preparation roadmaps
- Helps students prepare effectively for placements

## Features (Initial Phase)
- Career domain recommendation
- Resume keyword analysis
- Skill gap identification
- Preparation checklist

## Tech Stack (Planned)
- Frontend: HTML, CSS, JavaScript
- Backend: Python / Node.js
- AI Logic: Rule-based (Phase 1), ML-based (Future)
- Database: Firebase / MongoDB
